# iot-service-kit Github page

This directory contains the code for the IoT Service Kit site, http://futurice.github.io/cascade (also http://reactivecascade.com)

# Core Components

Gulp: Build and deployment script
SASS: CSS Preprocessor

# Development

## Setup

1. Go to build sub directory `cd build`

2. Install npm packages `npm install`

3. Run `gulp`.
